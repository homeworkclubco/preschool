Preschool is a minimal, extensible CSS framework and web component library designed for modern web applications.

  • TypeScript support with Lit framework
  • Customizable with CSS custom properties
  • Smooth animations with Web Animations API

Simply import the components and use them in your HTML. All components are framework-agnostic and work with vanilla JavaScript, React, Vue, or any other framework.
